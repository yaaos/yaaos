# core/webserver

> FastAPI app factory, lifespan, route registry, middleware, and SPA serving.

## Scope

- Owns: `RouteSpec` registry, `create_app()`, lifespan, middleware stack, SPA static serving, shutdown registries.
- Does NOT own: auth/CSRF/rate limiting (those are in [`core/auth`](core_auth.md)).
- `/api/health` is a framework carve-out (`core/webserver/health.py`) ‚Äî not registered via `RouteSpec`.

## Health endpoint (`/api/health`)

- Returns **200** when both `db_ok` and `redis_ok` are true; returns **503** when either is false.
- Body shape: `{status: "ok"|"degraded", db_ok: bool, redis_ok: bool, version: str}` ‚Äî unchanged regardless of status code.
- `version` is read from `settings.service_version` (`SERVICE_VERSION` env var, default `"0.0.0-dev"`). Set by the deploy pipeline (e.g. git SHA or semver tag).
- Exempt from Cloudflare ingress middleware; no auth required.
- Not traced ‚Äî the path is in `observability.TRACE_EXCLUDED_URLS`, passed to the FastAPI instrumentor, and the DB ping runs under suppressed instrumentation. See [core_observability.md](core_observability.md).

## Why / invariants

**One URL prefix per module** ‚Äî `register_routes(spec)` enforces: empty `router.prefix`, unique `module_name`, non-overlapping effective prefix, starts with `/api/`, no trailing `/`. Violations surface at import time. See `app/core/webserver/registry.py`.

**Lifespan order:**
1. Mount each router at its prefix.
2. Run `on_startup` hooks ‚Äî raising crashes boot (loud-by-design).
3. Mount SPA if `apps/web/dist` exists.
4. On shutdown: iterate `iter_web_shutdown_hooks()` in reverse registration order; errors logged + swallowed so all hooks run.

**Cache-Control on SPA files:**
- `/assets/*` ‚Üí `max-age=31536000, immutable` ‚Äî Vite content-hashes bundles; URL changes when content does.
- `index.html` + dist-root files ‚Üí `max-age=60, must-revalidate` ‚Äî not content-hashed; new deploys must be picked up quickly.

**Middleware stack** (outermost-first): OTel `FastAPIInstrumentor` ‚Üí request-ID middleware ‚Üí CORS (`["*"]` in dev; `YAAOS_CORS_ORIGINS` otherwise) ‚Üí unhandled-exception handler (500 JSON + log).

## Committed OpenAPI artifact

`apps/backend/openapi/web-api.json` is a committed, static snapshot of the full web API spec. It is generated by `apps/backend/bin/dump_web_openapi` (no DB/Redis/network required ‚Äî spec is built from the in-process FastAPI route registry alone). Backend CI fails if the artifact is stale.

The dump+strip logic lives in [`app/core/webserver/openapi_artifact.py`](../app/core/webserver/openapi_artifact.py); the drift gate is [`app/core/webserver/test/test_web_openapi_drift.py`](../app/core/webserver/test/test_web_openapi_drift.py).

**`/api/testing/*` paths are stripped** before writing ‚Äî those test-only backdoor endpoints must not appear in the committed artifact or be consumed by external tooling.

When you change the API shape (routes, response models, query params), run `apps/backend/bin/dump_web_openapi` and commit the updated artifact.

## Gotchas

- In dev (no `apps/web/dist`), non-`/api/` paths 404. Run `pnpm dev` separately.
- SPA catch-all guards path traversal via `relative_to(dist)`.
- `RouteSpec` is a Pydantic model; construction validates prefix rules before anything is registered.
- The OpenAPI artifact is generated under `APP_MODE=dev` (or `test` ‚Äî both work); the dump script sets no environment itself, so the caller must supply the required env vars (`DATABASE_URL`, `YAAOS_ENCRYPTION_KEY`, `REDIS_URL`, `YAAOS_PUBLIC_ORIGIN`). No actual connections are made.
