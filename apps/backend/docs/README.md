# Backend docs

FastAPI service in Python 3.13. Single Docker image runs the API, serves the bundled SPA, and spawns background coroutines for review work.

## Read first

- [modularity.md](modularity.md) ‚Äî Tach-enforced layer model (`core < domain < plugins < testing`), `RouteSpec` registry, module shape.
- [patterns.md](patterns.md) ‚Äî backend conventions: DI-over-patch, table-ownership, audit-log payloads, time-control env vars, async-everywhere, `spawn` contract.

## Directory shape

- `app/core/` ‚Äî infrastructure (no business logic).
- `app/domain/` ‚Äî business logic + plugin Protocols.
- `app/plugins/` ‚Äî vendor-specific implementations.
- `app/testing/` ‚Äî test-only scaffolding (excluded from prod wheel).
- `app/main.py` ‚Äî bootstrap composition order (load-bearing).
- `app/alembic/` ‚Äî hand-edited migrations using idempotent helpers.
- `bin/` ‚Äî `ci`, `sync_modules`, `check_table_access`.
- `conftest.py` ‚Äî pytest top-level fixtures.
- `pyproject.toml` ‚Äî uv + ruff config + TID251 bans.
- `tach.toml` ‚Äî generated by `bin/sync_modules`; never hand-edit.

## Running locally

`cd apps/backend && uv sync && uv run uvicorn app.main:app --reload --port 8080`. Docker runs: see [`docker/docker-compose.yml`](../../../docker/docker-compose.yml) and `docker-compose.test.yml`.

## Live API reference

FastAPI auto-generates an OpenAPI schema at `/openapi.json`. When the backend is running:

- **Swagger UI** ‚Äî `http://localhost:8080/docs`. Interactive endpoint browser; try-it-out forms.
- **Redoc** ‚Äî `http://localhost:8080/redoc`. Reference-style rendering of the same schema.

Both are derived from the live code ‚Äî always accurate, never written by hand. Hand-written docs here describe principles, architecture, and rules; the live reference is the authoritative endpoint catalogue.

## CI

`apps/backend/bin/ci` runs: `ruff format --check`, `ruff check`, `bin/sync_modules --check` (tach + layering), `bin/check_table_access`, `bin/check_doc_links` (cross-app docs), `pytest -q`, `semgrep scan` (rulesets `p/python` + `p/owasp-top-ten`; `.semgrepignore` at repo root excludes `app/testing/` + test files since they live outside the production wheel).
