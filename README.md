# yaaof

Yet Another Agent Orchestration Framework.

## Repo layout

- [`docs/`](docs/) — how the code works today.
- [`plan/`](plan/) — what we're building next.
  - [`VISION.md`](plan/VISION.md) — the whole product, end state.
  - [`ROADMAP.md`](plan/ROADMAP.md) — current milestone + backlog.
  - [`milestones/`](plan/milestones/) — one folder per milestone; each contains a `README.md` index plus its planning docs.
  - [`notes/`](plan/notes/) — scratchpad.
- [`CLAUDE.md`](CLAUDE.md) — working conventions for the assistant.
